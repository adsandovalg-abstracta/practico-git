# Workshop de Git

Bienvenido al repositorio base para el ejercicio práctico de capacitación en Git. Este es un entorno diseñado para que el equipo ponga en práctica los flujos de trabajo colaborativos utilizando Git y GitHub/GitLab.

---

## 📋 Instrucciones de la Tarea

Para completar el ejercicio práctico, sigue estos pasos:

1. **Crear una rama nueva**:
    Usa el prefijo `feat/` seguido de tu nombre y apellido.
    - Ejemplo: `feat/andres-sandoval`
2. **Preparar tu archivo**:
    Copia el archivo `docs/template-colaborador.md`, renómbralo a `tu-nombre-apellido.md` y muévelo a la carpeta `docs/v1/`.
3. **Realizar el Commit**:
    Asegúrate de seguir el estándar de [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/).
    - Ejemplo: `feat: agrega perfil y ejercicio de Andres Sandoval`
4. **Hacer Push y abrir Pull Request**:
    Sube tu rama al repositorio remoto y abre un PR solicitando la revisión de Andrés.

---

## ⚡ Comandos Rápidos

| Operación | Comando |
| :--- | :--- |
| Clonar Repo | `git clone [URL_DEL_REPO]` |
| Crear Rama | `git checkout -b feat/nombre-apellido` |
| Ver Estado | `git status` |
| Agregar Cambios | `git add .` |
| Realizar Commit | `git commit -m "feat: descripción corta"` |
| Subir Rama | `git push origin feat/nombre-apellido` |
| Actualizar Local | `git pull origin main` |
